Topic: Multi-label Classification and Regression (Updated: 2019)

In multi-label classification, multiple target variables are associated with each instance, as opposed to the traditional supervised learning problem where a single target variable (i.e., class) where only a single value is assigned to each instance. The main challenge is detecting and modelling dependencies among labels, while maintaining scalability to large problems.

Topic: Data Stream Classification (Update 2019)

Many real-world applications are found in the context of data streams, where data instances arrive rapidly and continuously in a theoretically-infinite stream, for example in sensor networks, online social media and text streams, anomaly and event detection. It is also relevant to reinforcement learning.

Many assumptions made in the literature about data streams don’t actually need to be met in practice and, on the other hand, many real constraints are not often taken into account, for example, we investigate:

  • Batch-incremental approximations are often good enough, or can be used together with incremental learning.

  • Supervision (true labels) may be delayed or often never available (semi-spervised/partially-labeled streams)

  • All concept-drifting streams contain temporal dependence, i.e., they are time series, and should be modeled and evaluated as such.

  • Hoeffding trees are not actually instance-incremental models, and in fact theoretically they can be seen as stable learners, unlike standard/batch decision tree induction. We can make them unstable (and thus better suited to ensembles) with streaming random patches.

We have started the Scikit-Multiflow project for learning from data streams in Python.

Application: Modelling and diagnosis of insomnia (Updated: 04/2019)

See slides, e.g., Machine Learning and AI in Healthcare.

We (work with Olivier Pallanca, neurophysiologist) are building predictive models for diagnosing different types of insomnia and, more importantly, predicting the response to different treatment options based on the personal characteristics of each patient. We have a large amount of data per patient (psychological questionnaires, overnight EEG and ECG signals, skin conductance, eye movement, reaction time, …). This involves a number of subtasks such as event detection in sequences and streams involving multiple correlated outputs. Interpretability of models is a key aspect.

Application: Modelling tree growth in Scots pine (2015)

We worked with forestry scientists from the University of Helsikni to model intra-annual growth of Scots pine trees at sites in Finland and France using time series and machine learning models (paper).

Kaggle Competition: Meta labels for Large Scale Hierarchical Text Classification (2014)

We won the 2014 LSHTC4 Kaggle competition, in part using an approach based on meta labels. Papers (see publications): - Multi-label Classification with Meta Labels - Kaggle LSHTC4 Winning Solution

Application: Tracking on very low-power sensor motes (2012)

We formulated and implementing a distributed particle filter on very low-power motes (4 MHz CPU) for real-time target tracking. This video of a demo of testbed deployment shows tracking using only light sensor observations.
